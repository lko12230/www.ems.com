Company Asset Policy

1. Hardware, Software, and Applications

  • All hardware, software, and applications must be approved and managed by IT.
  • Unauthorized installation or modification of software is strictly prohibited.
  • Company-owned assets must be properly documented and maintained.

2. Asset Maintenance and Security

  • All company assets require regular maintenance as per IT standards.
  • Assets should not be removed from company premises without prior approval.
  • All assets must be tagged with an official company asset label.

3. Mobile Device Usage

  • Personal devices must not be connected to the corporate network without approval.
  • Mobile devices accessing company data must comply with security protocols.
  • Encryption is mandatory for confidential data stored on mobile devices.

4. Data Backup & Media Handling

  • Backups must be conducted regularly in compliance with company policies.
  • Data stored on removable media should be encrypted and securely disposed of when no longer needed.
  • Loss or theft of media must be reported to IT immediately.
