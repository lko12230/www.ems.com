# 🚀 Employee Management System (EMS)

A modern **Employee Management System (EMS)** built with **Java Spring Boot** to simplify employee management, attendance tracking, leave management, task assignment, payroll, authentication, and performance monitoring.

---

## 📌 Features

### 👤 Employee Management
- Employee Registration
- Employee Profile Management
- Department & Team Management
- Role Based Access Control

### 🔐 Authentication & Security
- Spring Security Authentication
- Google OAuth Login
- GitHub OAuth Login
- Email OTP Verification
- Forgot Password
- Account Lock/Unlock
- Session Management
- CAPTCHA Protection

### 📅 Attendance Management
- Daily Attendance
- Monthly Attendance Report
- Attendance Calendar
- Working Hours Calculation
- Holiday Management

### 📝 Leave Management
- Apply Leave
- Leave Approval Workflow
- Leave History
- Leave Balance Management

### 📋 Task Management
- Assign Tasks
- Task Status Tracking
- Task Dashboard
- Due Date Monitoring
- Email Notification on Task Assignment

### 💳 Payment Integration
- Razorpay Payment Gateway
- Online Subscription
- Payment History
- Invoice Generation

### 📧 Email Services
- OTP Emails
- Task Assignment Emails
- Account Notifications
- Password Reset Emails

### 📊 Dashboard
- Employee Dashboard
- Admin Dashboard
- Attendance Statistics
- Leave Statistics
- Performance Analytics

### ⚙️ Scheduler
- Dynamic Job Scheduler
- Automatic Background Jobs
- Retry Mechanism
- Server Health Monitoring

### 📁 File Management
- Google Drive Integration
- File Upload
- Resume Upload
- Employee Documents

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/lko12230/www.ems.com.git
```

## Open Project

```bash
Import as Maven Project
```

## Configure Database

```
application-dev.properties
```

Update

```
spring.datasource.url
spring.datasource.username
spring.datasource.password
```

## Run

```bash
mvn spring-boot:run
```
